About project

This project gives ability to book company's resources for logged in workers. Users can do many operations, including:

  1. registration,
  2. log in,
  3. booking different resources,
  4. ability to resgin from users resources,
  5. ability to see users resources,

and many more. For now users can choose from three categories:

  1. parking places,
  2. books,
  3. trainings.
